Condicionales:

  • if: Código que se ejecuta si se cumple una condición.
  • else if: Código que se ejecuta si la condición inicial no se cumple, pero una segunda condición sí.
  • else: Código que se ejecuta en caso de que las demás no ocurran.

Únicamente es necesario definir if, los otros pueden omitirse si no se quiere ejecutar nada al no cumplirse la condición
